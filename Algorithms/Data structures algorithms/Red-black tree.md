```ad-attention
title: Restrições da estrutura

1. Cada nó tem uma cor: vermelho ou preto;
2. A cor da raiz é preto;
3. Não há nós vermelhos adjacentes;
4. Cada caminho desde um nó até qualquer um dos seus descendentes $NULL$ tem o mesmo número de nós pretos.
```

```ad-note
title: Complexidade
collapse: true

- Search/insert/remove: $O(\log n)$
- Find min/max key: $O(h)$
```
